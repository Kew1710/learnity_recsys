# Доступные команды (Skills)

| Команда | Описание |
|---|---|
| `/simplify` | Проверяет изменённый код на переиспользование, качество и эффективность, затем исправляет найденные проблемы. |
| `/loop` | Запускает промпт или команду с заданным интервалом (например, `/loop 5m /foo`, по умолчанию 10м). Для повторяющихся задач и периодического опроса. |
| `/claude-api` | Помогает создавать приложения с использованием Claude API или Anthropic SDK. Срабатывает при импорте `anthropic` / `@anthropic-ai/sdk`. |
| `/database-schema-designer` | Проектирует надёжные и масштабируемые схемы баз данных (SQL и NoSQL): нормализация, индексы, миграции, ограничения, оптимизация производительности. |
| `/gemini` | Запускает Gemini CLI для ревью кода, ревью планов или обработки больших контекстов (>200k токенов). По умолчанию использует Gemini 3 Pro. |
| `/draw-io` | Создание, редактирование и просмотр диаграмм draw.io: редактирование XML, конвертация в PNG, настройка макета, использование AWS-иконок. |
| `/find-skills` | Помогает найти и установить новые skills, когда нужна дополнительная функциональность или расширение возможностей. |
| `/command-creator` | Создаёт slash-команды для Claude Code с правильной структурой и лучшими практиками. |
| `/codex` | Запускает Codex CLI (codex exec, codex resume) для анализа кода, рефакторинга и автоматического редактирования. По умолчанию использует GPT-5.2. |
| `/agent-md-refactor` | Рефакторит перегруженные файлы `AGENTS.md`, `CLAUDE.md` и подобные: разбивает монолитные файлы на организованную документацию. |
| `/lesson-learned` | Анализирует недавние изменения кода через git-историю и извлекает уроки по разработке ПО. |
